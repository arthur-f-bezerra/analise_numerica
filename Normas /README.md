Normas é uma função que atribui um vetor para os reais, ou, || . || : V -> R

Para ser uma norma, precisa seguir as propriedades:

    (i) ||x|| >= 0 e ||x|| = 0 <-> x=o Sendo "o" o vetor nulo e x um vetor
    (ii) ||y * x|| = |y| * ||x|| Sendo y um escalar e x um vetor
    (iii) ||u + v|| <= ||u|| + ||v|| Sendo u, v vetores
